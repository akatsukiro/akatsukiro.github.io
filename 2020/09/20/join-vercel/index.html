使用Vercel来加速部署在Github的网站吧！
发表于2020-09-20|更新于2020-09-20|Vercel技术
|字数总计:636|阅读时长:1分钟|评论数:

前言

本站托管于Github，已使用Vercel加速托管。

听别的大佬说Vercel挺棒的，于是乎……

注册

  • 注册Github

最先注册的当然是Github啦！没有Github就没有接下来的事啦！
注意使用非QQ邮箱注册哦！Vercel不认QQ邮箱的。当然如果你已经注册了，那可以下一步了。如果你使用的是QQ邮箱注册的Github，那赶快将主邮箱换成非QQ邮箱哦！

  • 注册Vercel

进入官网
首先点击connect to GitHub。
如果已经登陆了GitHub的话应该会直接跳出你账号的登陆询问页面，一直点就够了！！

  • 注册域名

这个，找个自己喜欢的域名去注册即可。

部署

到了欢迎界面，Vercle会让你创建文件，填好后点击continue就好了
然后让你选择Import Git Repository或者Import Template

由于我博客早就部署在了GitHub，于是我选择了Import Git Repository，点击continue。
然后让你填写项目地址，去到博客的项目地址复制粘贴即可。
随后选则目录，默认即可，可以根据自己的需求更改。
之后填项目在vercel的名称，和项目使用的框架。由于我博客原先使用的就是Hexo，于是直接选择hexo框架。
等待vercel部署完成后就可享受极致访问速度了

新注册GitHub的童鞋当然是没有文件部署在了GitHub啦！所以选自右边的 Import Template，点击select template，数十种架构任君挑选！
在此我们选择Vercel的亲儿子，Next.js
然后让你选择Vercel账户，咱用自己的账户即可，选自自己的账户，点击continue。
下一页面没啥要改的话continue即可。
下一页导入nextjs模板，没啥要改的，deploy。
稍等一会，然后vercel就跑起来啦！总等待时间大约30s即可使用。然后你就可以开心建站啦！

绑定域名

点击进入项目，选择view domains，添加自己分给它域名，然后去DNS服务商处添加/更改Vercel提供的解析，即可使用自己的域名访问啦！
同时Vercel还贴心的免费帮你申请证书和续签，麻麻再也不用担心我域名没SSL啦！

完成

基本上就是如上。由于博主已经注册，所以注册相关教程说的比较简单。如果你发现文章有错误，欢迎回复。
